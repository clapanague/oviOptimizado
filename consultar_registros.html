Consultar Registros en Tablas

Para realizar consultas en una tabla utilizamos la sentencia SELECT cuya sintaxis es:
SELECT columna FROM nombreTabla

En este caso utilizaremos nuevamente un formulario de HTML para enviar la instrucción a la base de datos. En la imagen observamos la estructura del formulario y el código que ejecuta la consulta de los registros.
