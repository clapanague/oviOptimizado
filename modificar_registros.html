Modificar datos en la tabla

Para este OVI utilizaremos una de las formas para editar datos en un atabla en MySql. Utilizando un formulario realizamos el proceso de forma muy sencilla. Conociendo el Id del registro que vamos a editar, por ejemplo, si vamos a editar los datos de un empleado, debemos saber su id (clave primaria tabla empleado) que generalmente es su numero de identificación.

En la imagen observamos la estructura del formulario y el código que ejecuta para la edición de los registros.
